Security

Your data. Your keys. Our paranoia.

AES-256-GCM Encryption

All sensitive data — API keys, tokens, credentials — encrypted at rest with AES-256-GCM. Industry standard.

Metadata Only

We never store your code. Only package manifests, lockfile hashes, and app metadata. Your source stays on GitHub.

Minimum Scopes

GitHub OAuth requests only the scopes we need. Read-only repo access. No write permissions. No org data.

BYOK Architecture

Your LLM API keys are encrypted end-to-end. They never transit our servers unencrypted. You own the provider relationship.
